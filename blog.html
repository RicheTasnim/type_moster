1. Difference between localStorage and sessionStorage :

SessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

2. Difference between global scope and block scope :

In a browser environment, the global scope is controlled by the window object while in Node. js, it's controlled by the global object. 4. Why do we get undefined??

A variable that has not been assigned a value is of type undefined. A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned.
